# Pandabot

Evolução open-source do Takeshi-Bot em TypeScript, com Clean Code, SOLID, sistemas de aluguel/loja/VIP IAs.

## Setup
1. npm install
2. cp .env.example .env && edit keys
3. npm run db:migrate
4. npm start
